# GET Auction House

Semester Project 2 – Auction website built with Vanilla JavaScript, Bootstrap and Sass.

## Description

GET Auction House is a frontend auction application where users can browse listings, search, filter, view listing details, place bids, create listings and manage their profile.

## Built with

- HTML
- CSS / Sass
- Bootstrap
- Vanilla JavaScript
- Noroff API v2

## Features

- Browse auction listings
- Search listings
- Filter and sort listings
- View single listing pages
- Image gallery / carousel
- Register and log in
- Create new listings
- Place bids
- View profile listings and bids
- Update profile
- Responsive mobile layout
- Mobile bottom navigation
- Mobile filter modal

## Getting started

### Install

Clone the repo:
git clone YOUR_REPO_LINK_HERE

Install dependencies:
npm install

Run Sass watch
npm run watch

### Build Sass

npm run build
